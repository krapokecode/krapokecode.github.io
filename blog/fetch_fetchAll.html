มันเอาไว้ทำอะไร

ถ้าใครมาสาย Web programming จะรู้ว่าเอาไว้สำหรับดึงข้อมูลมาจาก DB นั้นเองแต่ถ้าใครสังเกตุสักนิดหนึ่งก็จะเห็นว่าทำไมมันต้องมี 2 ตัวทั้งที่ทำงานเหมือนกันทุกอย่าง เรามาดูกันก่อนว่ามันเขียนกันยังไง 

$sql = new PDO('mysql:host=localhost;dbname=test', $user, $pass);
$query = $sql->prepare('SELECT * FROM test1;');
$fetch = $query->fetch(PDO::FETCH_ASSOC);
foreach($data = $query->fetch)
{
echo $data
}
ถ้าเป็น fetchAll() ก็เปลี่ยนต้อง fetch เป็น fetchAll

แล้วมันต่างกันยังไง

สิ่งแตกต่างกันกันก็คือเจ้าตัว fetchAll() จะทำการนำข้อมูลออกมาเลยทีเดียวและใช้ระยะเวลา น้อยกว่า fetch() แต่ว่าการใช้ memory ของ fetchAll() จะกินเยอะกว่า fetch() อยู่ 2-3 เท่าตัว

สรุปจะใช้ตัวไหนดี
