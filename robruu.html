มันคืออะไรเกี่ยวกับอะไร

มันคือ Web application ที่ทุกคนสามารถแชร์ความรู้ลงได้ ไม่จำเป็นว่าจะต้องเป็นเกี่ยวกับความรู้ในห้องเรียนอย่างเดียวเป็นความรู้ทั่วไปที่สามารถใช้ในชีวิตประจำได้ด้วย

Feature

แบ่งเป็น 2 อย่างย่อยๆคือ ผู้สอนกับผู้เรียน
ผู้สอน :
        - อัพโหลด video ได้ 
        - สร้างโจทย์ พร้อมคำตอบ 4 ตัวเลือก
        - สร้างเนื้อหาเป็นสรุปหรือตัวอย่างโจทย์ได้โดยใช้ editor บนเว็บ
ผู้เรียน :
        - ตั้งกระทู้สอบถามในคอร์สเรียน
        - ตอบคำถามต่างๆภายในคอร์สเรียนเพื่อนำคะแนนไปแลกซื้อบทเรียนอื่น
        - สร้างโน๊ตสรุปในบทเรียน

Frontend

   - ใช้ bootstrap เป็นหลัก 
   - ใช้ ckeditor สำหรับทำ editor content
   - ใช้ jquery ทำเกี่ยวกับ ajax แต่อีกไม่นานคงจะได้ไปใช้ angular 2 

Backend 

   - architect เป็น monolith แต่คงจะเปลี่ยนเป็น microservice
   - Web service ใช้เป็น Apache แต่อีกไม่นานคงจะเปลี่ยนเป็น Nginx เพราะรองรับ concurrent ได้เยอะกว่า
   - ตัวภาษายังใช้เป็น PHP7 
   - RDBMS เป็น mysql เพระต้องทำเกี่ยวกับ transaction ในอนาคต

Q&A

  Q : ทำไมถึงสร้างระบบนี้ขึ้นมา
  A : ผมอยากจะมี social network ที่เอาไว้แชร์ความรู้จริงๆโดยไม่มีสิ่งอื่นมาเกี่ยวข้อง

  Q : ฟรีรึเปล่า
  A : ถ้าใครต้องการโดยไม่แสวงหาผลกำไรก็สามารถทำได้ แต่ใครที่ต้องเอาเพื่อหาประโยชน์ส่วนตัวต้องมาคุยกันก่อน

  Q : ถ้าอยากจะมีส่วนร่วมการทำสร้างจะได้มั๊ย
  A : ได้แน่นอนครับ สามารถติดต่อถ้า FB ได้ครับ
