What is Fastify

        " Fast and low overhead web framework, for Node.js "

       Fastify is a node framework that can get the request from the client more than the other framework. it was designed for scaling and lower cost of the infrastructure, so it is the best for the developer experience with the least overhead and a powerful plugin architecture. It is inspired by Hapi and Express, so It familiar in Express API

Pros

- scaling

- creating plugin

- more connection


Cons

- Community & Ecosystem

- Few plugins
